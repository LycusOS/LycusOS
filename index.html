Lycus OS

Just another arch based distro by Sarvesh Kardekar.

  • Learn more

What the hell is this?

Lycus OS is an Arch based distro with a focus on minimalism and stability. It uses BSPWM as the default window manager and a fork of picom as the compositor.

What do I get?

You get a minimal, beautiful and functional system whcih has been made possible by several open source softwares.

BSPWM

BSPWM or Binary Space Partition Window Manager is a minimalistic modern window manager that represents windows as the leaves of a full binary tree.

Polybar

Polybar is a fast and easy-to-use tool for creating status bars. It aims to help users build highly customizable status bars, without the need of having a black belt in shell scripting.

Picom

Picom is a standalone compositor for Xorg, suitable for use with window managers that do not provide compositing.

Systemd-Boot

Systemd-boot is a simple UEFI boot manager which executes configured EFI images.
